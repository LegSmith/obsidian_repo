  

# 회원 등록

---

- MemberForm 클래스는 화면등록 화면에서 데이터를 가져오는 DTO이다. 
    
    ```java
    @Getter @Setter
    public class MemberForm {
    
        @NotEmpty(message = "회원 이름은 필수 입니다.") // Validation을 해준다.
        private String name;
    
        private String city;
        private String street;
        private String zipcode;
    }
    ```
    
- @NotEmpty 어노테이션을 사용함으로써 나중에 Validation을 해준다.
- MemberController에서는 아래와 같이 작성한다.
- 만약 회원등록 화면에서 이름을 정하지 않으면 오류가 생긴다.
- 오류 내용을 BindingResult 를 이용하여 오류내용을 result에 담아두고 result로 오류를 제어한다.
- ==**Server Side Rendering에서 오류를 제어하는건 BindingResult이다.**==
    
    ```java
    @Controller
    @RequiredArgsConstructor
    public class MemberController {
    
        private final MemberService memberService;
    
        @GetMapping("/members/new")
        public String createForm(Model model){
            model.addAttribute("memberForm",new MemberForm());
            return "members/createMemberForm";
        }
    
        @PostMapping("/members/new")
        public String create(@Valid MemberForm form, BindingResult result){
            //@Valid : MemberForm 에 있는 @NotEmpty 가 붙은 컬럼을 Validation 을 해준다.
            //BindingResult : 오류가 생기면 오류내용이 result 에 담겨서 사용 가능하다.
    
            if(result.hasErrors()){ //오류가 있다면
                return "members/createMemberForm";
            }
    
            Address address = new Address(form.getCity(), form.getStreet(), form.getZipcode());
    
            Member member = new Member();
            member.setName(form.getName());
            member.setAddress(address);
    
            memberService.join(member);
    
            return "redirect:/";
    
        }
    }
    ```
    

# 상품 수정( 매우 중요 !! )

---

- `@PathVariable` 어노테이션을 이용하여 id값을 가진채로 상품수정 페이지로 이동한다. 이때 상품수정페이지에서 보이는 정보들은 BookForm객체의 데이터들이다.
- 수정하기(Edit)버튼을 누르면 Post로 updatItem() 로직이 실행되고, BookForm객체를 받는다.(`@ModelAttribute` 어노테이션을 붙여준다.)
    - `@ModelAttribute` : Request 로 전달하는 값을 오브젝트 형태로 매칭해주는 어노테이션
- 받은 BookForm객체를 다시 Book엔티티로 바꿔서 save해준다.

```java
//상품 수정
    @GetMapping("items/{itemId}/edit")
    public String updateItemForm(@PathVariable("itemId") Long itemId, Model model){
        Book item = (Book)itemService.findOne(itemId); //ㅇㅖ제에서는 무조건 Book만 받음

        BookForm form = new BookForm();
        form.setId(item.getId());
        form.setName(item.getName());
        form.setPrice(item.getPrice());
        form.setStockQuantity(item.getStockQuantity());
        form.setAuthor(item.getAuthor());
        form.setIsbn(item.getIsbn());

        model.addAttribute("form",form);

        return "items/updateItemForm";
    }

    @PostMapping("items/{itemId}/edit")
    public String updateItem( @PathVariable String itemId, @ModelAttribute("form") BookForm form){

        Book book = new Book();

        book.setId(form.getId());
        book.setName(form.getName());
        book.setPrice(form.getPrice());
        book.setStockQuantity(form.getStockQuantity());
        book.setAuthor(form.getAuthor());
        book.setIsbn(form.getIsbn());

        itemService.saveItem(book);

        return "redirect:/items";

    }
```
