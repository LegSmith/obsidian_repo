  

## 스프링 메시지 소스 설정

---

- `MessageSource` 를 스프링 빈으로 등록하면 된다.
- `MessageSource` 는 인터페이스이기 때문에 `ResourceBundleMessageSource` 구현체를 스프링 빈으로 등록하면 된다.
    
    - setBasename() 메서드에 들어오는 이름으로 properties 를 읽어 들인다.
    
    ```java
    @Bean
      public MessageSource messageSource() {
          ResourceBundleMessageSource messageSource = new
      ResourceBundleMessageSource();
          messageSource.setBasenames("messages", "errors");
          messageSource.setDefaultEncoding("utf-8");
          return messageSource;
    }
    ```
    

### 스프링 부트 메시지 소스 설정

- 스프링 부트는 위와 같이 스프링 빈으로 등록할 필요가 없다.
- `application.properties` 에 추가해주면 된다.
    
    ```xml
    spring.messages.basename=messages,config.i18n.messages
    ```
    

## 메시지 파일 만들기

---

- `resources/` 경로 밑에 `properties` 파일을 만들면 된다.
- `message.properties`
    
    ```xml
    hello=안녕
    hello.name=안녕 {0}
    ```
    
- `message_en.properties`
    
    ```xml
    hello=hello
    hello.name=hello {0}
    ```
    
- 미국에서 들어올 땐 `message_en.properties` 가 읽혀서 영어로 나오고 미국 외에는 `message.properties` 파일이 읽혀서 ‘안녕’ 이 출력된다.

## 메시지 소스 사용하기

---

- 가장 단순한 메시지 코드로 사용하기
- `getMessage()` : 스프링 빈에 등록되어있는 MessageSource 에 있는 메서드이다.
    - `code` : 첫번째 파라미터, properties에 적혀있는 key 값이다.( 예시→ `hello = 안녕` 에서 hello)
    - `args` : properties 에 있는 `hello = 안녕 {0}` 부분에 바인딩 되는 값이다.
    - `locale` : 언어가 들어간다
- `getMessage()` 에 있는 code가 properties에 없으면 `NoSuchMessageException` 예외를 터뜨린다.
    - `code` 가 없으면 `DefaultMessage` 를 지정해줄 수 있다.
- argument 를 직접 넣어줄 수 있다.
    - `getMessage("hello.name", new Object[]{data}, null)`
- 국제화를 사용할려면 `locale` 옵션을 추가해주면 된다.
    
    ```java
    @SpringBootTest
    public class MessageSourceTest {
    
        @Autowired
        MessageSource messageSource;
    
        @Test
        public void helloMessage() throws Exception {
            String result = messageSource.getMessage("hello", null, null);
            assertThat(result).isEqualTo("안녕");
        }
    
        @Test //code 가없으면 예외
        void notFoundMessageCode(){
            assertThatThrownBy(()->messageSource.getMessage("no_code",null,null))
                    .isInstanceOf(NoSuchMessageException.class);
        }
    
        @Test // code가 없을때 디폴트메시지 설정
        void notFoundMessageCodeDefaultMessage(){
            String result = messageSource.getMessage("no_code", null, "기본 메시지", null);
            assertThat(result).isEqualTo("기본 메시지");
        }
    
        @Test // argument 세팅
        void argumentMessage(){
            String message = messageSource.getMessage("hello.name", new Object[]{"Cristiano","I'm Messi"}, null);
            assertThat(message).isEqualTo("안녕 Cristiano,I'm Messi");
        }
    
        @Test // 기본 국제화
        void defaultLang(){
            assertThat(messageSource.getMessage("hello",null,null)).isEqualTo("안녕");
            assertThat(messageSource.getMessage("hello",null, Locale.KOREA)).isEqualTo("안녕");
        }
    
        @Test // 영어권 국제화
        void enLang(){
            assertThat(messageSource.getMessage("hello",null, Locale.ENGLISH)).isEqualTo("hello");
        }
    }
    
    @Test
    void helloMessage() {
    	String result = ms.getMessage("hello", null, null);
    	assertThat(result).isEqualTo("안녕");
    }
    ```
    

## 애플리케이션에 메시지 적용하기

---

- 메시지를 미리 `key=value` 형식으로 `message.properties`에 등록해준다.
    
    ```xml
    label.item=상품
    label.item.id=상품 ID
    label.item.itemName=상품명
    label.item.price=가격 
    label.item.quantity=수량
    
    page.items=상품 목록 
    page.item=상품 상세 
    page.addItem=상품 등록 
    page.updateItem=상품 수정
    
    button.save=저장
    button.cancel=취소
    ```
    
- 그리고 사용자에게 보여질 View에서 th:text 속성을 사용하여 #{key} 를 넣어주면 된다.
    - `<h1>상품</h1>` 이라면 `<h1 th:text=”#{label.item}”>상품</h1>` 으로 수정해주면 된다.
    - 추후에 기획자가 ==**_애플리케이션에서 상품이란 단어를 아이템으로 바꾸자고 하면_** `message.properties` **_에서만 수정해주면 애플리케이션 전체에 적용된다._**==
- 만약 파라미터가 있는 key 라면
    
    - `hello.name=안녕 {0}` → `<p th:text="#{hello.name(${item.itemName})}"></p>`
    
    ### 국제화
    
    - 국제화는 브라우저 설정언어에 따라 자동으로 바껴지기 때문에 `message_en.properties` 에 적어주기만 하면 된다 !!
        
        ```xml
        hello=hello
        hello.name=hello {0},{1}
        
        label.item=Item
        label.item.id=Item ID
        label.item.itemName=Item Name
        label.item.price=price
        label.item.quantity=quantity
        
        page.items=Item List
        page.item=Item Detail
        page.addItem=Item Add
        page.updateItem=Item Update
        page.saveStatus=Save Complete
        
        button.save=Save
        button.cancel=Cancel
        ```
