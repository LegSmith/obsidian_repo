## 1. OSI 7 Layer 와 식별자

### 1.1. OSI 7 Laye의

- 가장 아래 계층을 1계층이라 하고 제일 위는 7계층이면서 사용자와 직접 상호작용하는 계층이다.
- ==**물리계층 → 데이터 링크 계층 → 네트워크 계층 → 전송계층 → 세션계층 → 표현계층 → 응용계층**==
- 결국엔 H/W 계층에 NIC 이 있고 바로 위에 구동하는 Driver가 있고, Kernel mode 에는 TCP/IP 프로토콜이 존재하고 User mode 에는 Chrome 같은 브라우저를 이용하여 HTTP 통신을 한다.

    ![[image 19.png|image 19.png]]

- 그래서 5,6,7 계층이 User mode 에 Chrome 같은 브라우저를 담당하고 있고 4계층이 Kernel mode 에 TCP, 3계층이 IP 그리고1,2 계층이 Driver 와 NIC을 담당한다고 보면 된다.

    ![[image 1 10.png|image 1 10.png]]


### 1.2. 식별자

- L2 에서 NIC 에 식별자는 MAC 주소이다.
    - 컴퓨터에 NIC이 3개 꽂혀있다면 MAC 주소는 3개이다.
- L3 에서 식별자는 IP 주소이다.
    - Ipv4 와 Ipv6가 있다.
- L4 에서 식별자는 Port 번호이다.

### 1.3. ipconfig 명령어

- `Windows` 에서 명령프롬프트 창에서 `ipconfig /all` 명령어를 쳐서 내용을 알아야 한다.
    - `win + R` → `cmd` 입력
- `Linux` 에서는 `ifconfig` 이다.
- 물리적 주소가 MAC 주소이다.

    ![[image 2 8.png|image 2 8.png]]


가장 중요한 것은 ==**식별자**== 이다.

- **MAC 주소**는 `NIC` 을 식별한다.
- **IP 주소**는 `Host` 를 식별한다.
- **Post 번호**는 `Process` or `L2 인터페이스` or `Service` 일 수 있다.

## 2. Host 를 외우는 방법

- Host 는 Computer 에 Network 가 연결된 것을 말한다.
- 이 때, Host 를 지칭할 때 End-point 와 Switch 로 나뉜다.

    - `Switch` : Network 그 자체를 이루는 Host
        - 가장 대표적으로 `Router` 가 있다.
        - 그 외에도 `IPS` 가 있는데, 이 때는 `Security Switch` 라 한다. 이 Switch 는 **보안목적**이다.
        - `Tab Switch` , `Aggregation Switch` 가 있는데 이 Switch 들은 **관리목적**이다.
    - `End-point` : Network 를 이용하는 주체 이다. 직역하면 **단말기** 이다.
        - 대표적으로 `Client` 과 `Server` 가 존재한다. 즉, PC 를 말한다.
        - P2P 통신할때 Peer 라 하는데, Peer 도 End-point 이다.

## 3. Switch 가 하는 일과 비용

### 3.1. Switchin의

- 쉽게 비유를 할 때 자동차가 갈림길에서 이정표를 보고 목적지로 가는 것을 비유로 들 수 있다.
    - 자동차 : `Packet`
    - 길 : `Network`
    - 교차로 : `L3 Switching` = `Router`
    - 이정표 : `Routing Table`
- 이 때 **경로를 선택하는 것을 Interface 를 선택**하는 것이다. 이를 `Switching` 이라 한다.
- 이 때 목적지인 **==IP주소를 근거로 경로==**를 `Switching` 한다.
- 각 Router 마다 Routing Table 이 있는데, 이는 표(Table)이다.
    - 만약 MAC 주소로 Switching 하면 L2 Switching 이다.
    - 만약 Port 번호로 Switching 하면 L4 Switching 이다.
    - HTTP 프로토콜 근거로 Switching 하면 L7 SWitching 이다.

### 3.2. 비용

- Switching 을 할 때는 적은 비용으로 목적지로 가야 한다.
- 이 때 비용을 `Matric` 이라 부른다.
- 즉, Matric 값이 낮은 경로로 이동을 한다.

- CMD 창을 열고 `route print` 를 입력하면 ==**인터페이스 목록**==, ==**라우팅 테이블**== 목록이 나온다.

    - 이 때 메트릭(Matric)이 Switching 비용이다.
