### 3.2. 가상 어댑터

![[image 9 6.png|image 9 6.png]]

  

- `Host OS`(Visual Studio가 돌아가는 OS), `Guest OS`(Virtual Machine)가 서로 통신할 때 ==**L2 스위치로 LAN 케이블 연결하듯이 통신을 하는데 이 떄 사용되는데 VM어댑터1 번**==이다.
- 즉, `VM어댑터1`번을 통해 ==**Host OS 와 Guest OS 의 통신내을 알 수 있다.**==
- `VM어댑터8`번은 Host OS 입장에서 보면 단순히 NIC 드라이버로 보이지만 ==**공유기 역할**==을 한다.

### 5. Visual Studio 설정

- 우선 npcap sdk 파일 을 C 드라이브 아래에 npcap 이라는 이름으로 저장했을 것이다.
- Visual Studio 를 실행하고 `Alt + F7` 을 누른다.
- VC++ 디렉터리를 누르고 외부 include 디렉터리와 라이브러리 디렉터리에 다음과 같이 경로를 추가한다.
    
    - 외부 include 디렉터리 : `C:\npcap\Include`
    - 라이브러리 디렉터리 : `C:\npcap\Lib\x64`
    
    ![[image 12 4.png|image 12 4.png]]
    
- 그리고 링커 → 입력을 들어가서 지연 로드된 DDL 에 `wpcap.dll` 을 추가해준다.
    
    ![[image 13 4.png|image 13 4.png]]

### 8. CLion 컴파일러 설정

보통 MinGW(gcc 기반) 컴파일러를 쓰기위해 CLion 을 쓰지만, 현재 Npcap 라이브러리가 MSVC 컴파일러 호환용으로 받았기 때문에 MSVC 컴파일러로 설정한다.

1. 설정(`ctrl + alt + s`) → `Build, Execution, Deployment` → `Toolchains`
2. `+` 눌러서 Visual Studio 선택 → Toolset 에 Visual Studio 2022 Community 가 설치되어 있는 경로로 설정
