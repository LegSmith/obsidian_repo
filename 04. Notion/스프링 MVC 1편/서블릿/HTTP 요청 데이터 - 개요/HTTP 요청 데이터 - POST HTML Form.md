HTML의 Form을 이용해서 클라이언트에서 서버로 데이터를 전송한다. 주로 회원가입, 상품 주문 등에서 사용하는 방식이다.


## 특징

- ==content-type : application/x-www-from-urlencoded==
- 메세지 바디에 **==username=hello&age=20==** 데이터를 전송(쿼리스트링이랑 비슷하다)


> **application/x-www-from-urlencoded 방식은 쿼리 파라미터 방식과 같다!!**
