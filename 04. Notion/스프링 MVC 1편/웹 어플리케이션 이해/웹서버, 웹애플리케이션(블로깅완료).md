# 웹서버(Web Serve)

- HTTP 기반으로 동작
- 정적 리소스 제공
- ==**정적 파일**==
- Apache

# 웹 애플리케이션 서버(WAS)

- HTTP 기반
- 웹서버 기능 포함 +
- ==**프로그램 코드를 실행해서 애플리케이션 로직 실행**==
    - ==**동적 HTML, HTTP API(JSON)**==
    - ==**서블릿, JSP, 스프링 MVC**==
- Tomcat , Jetty, Undertow


- WAS가 너무 많은 역할을 담당, 서버 과부하 우려
