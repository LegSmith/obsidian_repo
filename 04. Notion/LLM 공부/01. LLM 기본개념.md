

## 1.1. LLM 이란 무엇인가?
> LLM 은 대규모 데이터에서 훈련된 많은 수의 파라미터를 가진 신경망
- **신경망**이란 많은 머신러닝 알고리즘 중 하나로 텍스트, 이미지와 같은 데이터를 처리하는데 뛰어나다
- **파라미터**는 신경망 모델이 가진 노브(knob) 또는 다이얼이다. 파라티머가 많은수록 복잡한 패턴을 감지하는데 유리하다.
	- 노브(knob)란 ? 라디오나 오디오 기기에 달린 손잡이 다이얼을 뜻한다. 마치 다이얼 하나 처럼 특정 값으로 맞춰줘있다는 비유다.
- _신경망=딥러닝_
> LLM 은 `트랜스포머(transformer)`라는 구조의 신경망을 사용
- 트랜스포머는 기계 번역을 위해 고안된 신경망 구조이다.

### 인공지능, 머신러닝, 딥러닝과 LLM 사이의 관계

![[Pasted image 20260717123934.png]]

- _대규모 언어 모델(LLM) 모든게 트랜스포머 구조는 아니다._ 하지만 요새 나오는 모든 LLM은 트랜스포머 구조를 사용하고 있다.
- `GenAI`는 생성형 AI라고해서 심층 신경망을 사용하여 텍스트, 이미지 또는 다양한 형태의 미디어와 같은 새로운 콘텐츠를 생성한다.
	- **텍스트를 생성하는 모델은 트랜스포머(Transformer) 구조**를 사용한다.
	- **이미지를 생성하는 모델은 디퓨전(Diffusion) 구조**를 사용한다.

### 지도 학습과 비지도 학습
> 머신러닝 모델은 크게 지도 학습과 비지도 학습 방법으로 훈련한다.
- **지도 학습**은 _입력과 정답(타깃, 레이블)을 사용해 모델이 둘 사이의 패턴(수학 함수)를 학습_한다.
	- 예시로 _이메일 내용(입력)과 정답(스팸여부)를 사용해 스팸 필터 모델을 훈련_
- **비지도 학습**은 정답이 없이 입력만 사용하여 모델을 훈련한다.(차원 축소, 시각화 등)
	- 차원축소는 *데이터의 특성(변수) 개수를 정보 손실 최소화하면서 줄이는 것*
	- 시각화는 *축소된 차원을 사람이 눈으로 볼 수 있게 그래프로 그리는 것.*

## 1.2.  LLM 애플리케이션

1. **텍스트 분류 및 요약** : `BERT`, `RoBERTa(분류)` / `BART`, `T5`, `Pegasus(요약)` 등
	- 구글, 빙(bing)과 같은 검색 엔진의 결과를 개선
	- 범용 또는 전문적인 텍스트 데이터에서 지식을 추출
2. **사용자의 지시를 따라 응답** : `Gemini(Google)`, `ChatGPT(OpenAI)`, `Claude(Anthropic)` 등과 같은 챗봇 또는 가상 비서
3. **언어 번역이나 언어 교육** : `Google Translate`, `DeepL`, `GPT-4` 등
	- 프로그래밍 코드도 작성
4. **주어진 이미지를 이해하고 응답에 참고** : `GPT-4V`, `Gemini`, `Claude(Vision)` 등
	- 이미지 캡셔닝, 이미지 설명하기 등
	- **멀티 모달 LLM**이란? *텍스트뿐 아니라 이미지, 음성 등 여러 형태(모달리티)의 데이터를 함께 입력받고 처리할 수 있는 LLM*
